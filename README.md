# Presupuesto Mensual 21

Aplicación de gestión de presupuesto personal mensual.

## Despliegue en Netlify

Esta es una aplicación web estática sin proceso de build.

### Archivos principales:
- `index.html` - Página de inicio/landing
- `login.html` - Autenticación
- `app.html` - Aplicación principal
- `js/` - Módulos JavaScript
- `css/` - Estilos
- `assets/` - Imágenes y recursos

### Sin proceso de build
Esta aplicación usa:
- HTML puro
- JavaScript vanilla
- Tailwind CSS vía CDN
- No requiere npm install ni build

Simplemente despliega los archivos tal cual.
